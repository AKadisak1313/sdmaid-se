<i>SD Maid 2/SE</i> آپ کے Android کا بھروسہ مند معاون ہے، اسے صاف ستھرا رکھنے کے لیے۔

کوئی بھی کامل نہیں ہے اور نہ ہی Android ہے۔
* وہ ایپس جنہیں آپ پہلے ہی ہٹا چکے ہیں کچھ پیچھے چھوڑ جاتی ہیں۔
* لاگز، کریش رپورٹس اور دیگر فائلیں جن کی آپ کو ضرورت نہیں ہے وہ مسلسل بنتی رہتی ہیں۔
* آپ کا اسٹوریج ایسی فائلیں اور فولڈرز جمع کر رہا ہے جنہیں آپ نہیں پہچانتے۔
* آپ کی گیلری میں ڈپلیکیٹ تصاویر۔

آئیے یہاں مزید بات نہ کریں... SD Maid 2/SE کو آپ کی مدد کرنے دیں!

SD Maid 2/SE ایک ایپ اور فائل مینیجر ہے جو یہ جاننے میں مہارت رکھتا ہے کہ آپ کے آلے پر کن ایپس نے مخصوص فائلیں بنائی ہیں۔ SD Maid 2/SE آپ کے آلے کو تلاش کرتا ہے اور اسٹوریج کی جگہ کو محفوظ طریقے سے خالی کرنے کے اختیارات پیش کرنے کے لیے انسٹال کردہ ایپس کے ساتھ فائلوں کا موازنہ کرتا ہے۔

✨ ایپس کو اَن انسٹال کرنے کے بعد صفائی کریں
اگر ایپس اپنے مخصوص فولڈرز سے باہر فائلیں بناتی ہیں تو ایپس کو اَن انسٹال کرنے کے بعد بھی فائلیں باقی رہ سکتی ہیں۔ "CorpseFinder" ٹول ایپ کی باقیات کو تلاش کرتا ہے، آپ کو بتاتا ہے کہ وہ کس ایپ سے تعلق رکھتی تھیں اور انہیں حذف کرنے میں آپ کی مدد کرتا ہے۔

🔍 اپنے آلے کو ہوشیار طریقے سے تلاش کریں
خالی فولڈرز، عارضی فائلوں، پہلے سے انسٹال کردہ ایپس اور مزید کے لیے فلٹر کریں۔ آپ اپنے تلاش کے معیار بھی بنا سکتے ہیں۔ "SystemCleaner" ٹول آپ کو خود بخود اپنے آلے کو تلاش کرنے اور مختلف معیارات کی بنیاد پر فائلوں کو حذف کرنے کی اجازت دیتا ہے۔

🧹 قابل خرچ فائلیں اور پوشیدہ کیشے حذف کریں
تھمب نیلز، ردی کی ٹوکری، آف لائن کیشے اور مزید: اگر ایپس خود صفائی نہیں کرتی ہیں، تو یہ ایپ کرے گی۔ "AppCleaner" ٹول قابل خرچ فائلوں والی ایپس کو تلاش کرتا ہے۔

📦 اپنی تمام ایپس کا نظم کریں
اپنے آلے پر انسٹال کردہ تمام ایپس کی ایک جامع فہرست حاصل کریں۔ فعال، غیر فعال، صارف یا سسٹم ایپ: کوئی بھی ایپ آپ سے چھپ نہیں سکتی۔ "AppControl" ٹول ایک ایپ مینیجر ہے جو آپ کو اپنی ایپس کو تلاش کرنے، ترتیب دینے اور کنٹرول کرنے کی اجازت دیتا ہے۔

📊 آپ کی تمام جگہ کیا استعمال کر رہا ہے
فون اسٹوریج، SD کارڈز، اور USB آلات پر ایپس، میڈیا، سسٹم، اور دیگر فائلوں کے ساتھ اسٹوریج کا نظم و نسق پیچیدہ ہو سکتا ہے۔ "StorageAnalyzer" ایک فائل مینیجر ہے جو یہ دکھاتا ہے کہ آپ کے آلے پر جگہ کیسے استعمال ہوتی ہے، آپ کے اسٹوریج کے نظم و نسق کو آسان بناتا ہے۔

📷 ڈپلیکیٹ ڈیٹا تلاش کریں
ڈپلیکیٹ ڈاؤن لوڈز، سوشل میڈیا کے ذریعے بھیجی گئی تصاویر یا ایک ہی منظر کی ملتی جلتی تصاویر: وقت گزرنے کے ساتھ نقول جمع ہو سکتی ہیں۔ "Deduplicator" ٹول ایسی فائلوں کو تلاش کرتا ہے جو بالکل ایک جیسی یا ملتی جلتی ہوں اور اضافی نقول کو حذف کرنے میں آپ کی مدد کرتا ہے۔

یہ ایپ اشتہار سے پاک ہے۔ کچھ خصوصیات کے لیے بامعاوضہ اپ گریڈ کی ضرورت ہے۔

SD Maid 2/SE، SD Maid 1/Legacy کا جانشین ہے۔
نئے Android ورژنز کے لیے بہتر بنایا گیا ہے اور صفائی پر مرکوز ہے۔

اس ایپ میں اختیاری خصوصیات ہیں جو تکلیف دہ کارروائیوں کو خودکار کرنے کے لیے AccessibilityService API کا استعمال کرتی ہیں۔
AccessibilityService API کا استعمال کرتے ہوئے، یہ ایپ آپ کے لیے بٹن دبا سکتی ہے تاکہ آپ متعدد ایپس پر آپریشنز انجام دے سکیں، مثلاً کیشے حذف کرنا۔
یہ ایپ معلومات جمع کرنے کے لیے AccessibilityService API کا استعمال نہیں کرتی ہے۔

SD Maid 2/SE ایک فائل مینیجر اور کلینر ایپ ہے۔
